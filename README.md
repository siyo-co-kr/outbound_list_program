# 🏥 아웃바운드 양식 자동 필터링 도구
이 프로그램은 환자 정보가 담긴 엑셀 데이터를 불러와 내원일자, 생년월일, 연락처 유효성을 기준으로 데이터를 정제하고, 마케팅이나 안내 문자 발송에 적합한 최종 양식으로 변환해 주는 Python 기반 데스크톱 애플리케이션입니다.

---

## 🚀 주요 기능
1. **암호화 엑셀 지원**: 비밀번호가 설정된 엑셀 파일도 프로그램 내에서 즉시 복호화하여 읽을 수 있습니다.
2. **기간별 내원일 필터링**: 마지막 내원일 기준으로 '1개월'부터 '5년'까지 유동적인 기간 필터를 제공합니다.
3. **생년월일(나이) 필터링**: 특정 연령대를 타겟팅할 수 있도록 생년월일 범위 지정 필터를 지원합니다.
4. **연락처 자동 정제**:
   - 숫자 이외의 특수문자(하이픈 등) 자동 제거
   - 10자리 번호(010 제외 형태)를 11자리(010 시작)로 자동 보정
   - 중복 연락처 제거 및 유효한 휴대폰 번호 형식만 추출
5. **표준 양식 출력**: 정제된 데이터를 `차트번호(공란)`, `이름`, `연락처` 형태의 깔끔한 엑셀 파일로 저장합니다.
6.

## 🛠 실행 환경 및 설치 방법
1. 요구 사항
    - 개발 환경 : Python 3.12.10
    - 최소 요구 사양 : Python 3.8 이상
    - 필수 라이브러리: `pandas`, `PySide6`, `msoffcrypto-tools`, `openpyxl`, `pyinstaller`
2. 라이브러리 설치
  ```
pip install pandas PySide6 msoffcrypto-tools openpyxl pyinstaller
```
3. 실행 및 빌드 방법
   - **소스 코드 실행**: `python main.py`
   - **실행 파일(EXE) 생성**:
     ```
     pyinstaller --noconfirm --onefile --windowed --name "아웃바운드_필터_도구" main.py
     ```
---

## 📖 사용 방법
1. **파일 선택**: [파일 찾기] 버튼을 눌러 원본 환자 정보 엑셀 파일을 선택합니다.
2. **비밀번호 입력**: 엑셀 파일에 암호가 걸려 있는 경우 입력합니다. (없으면 공란)
3. **필터 설정**:
   - **내원일**: 원하는 기간(예: 6개월 이내)을 선택합니다.
   - **생년월일**: 특정 연령대 추출이 필요하면 '필터 사용' 체크 후 날짜 범위를 설정합니다.
4. **데이터 정제 및 저장**: [데이터 정제 및 저장] 버튼을 누르면 필터링된 결과가 새 엑셀 파일로 저장됩니다.

---

## ⚠️ 데이터 요구 양식
프로그램이 정상 작동하려면 원본 엑셀 파일(**두 번째 시트**)에 아래와 같은 정확한 컬럼명이 포함되어야 합니다.

| 컬럼명 | 필수여부 | 설명 |
| :--- | :---: | :--- |
| **차트번호** | 필수 | 존재 여부 확인용 (출력 시 공란 처리) |
| **이름** | 필수 | 환자 성함 |
| **연락처** | 필수 | 휴대전화 번호 (010 형식으로 자동 보정) |
| **마지막 내원일자** | 필수 | 최종 방문일 (내원일 필터 기준) |
| **생년월일** | 선택 | 생년월일 필터 사용 시 필수 |

---

## 💡 참고 사항
- **중복 제거**: 동일한 연락처가 있을 경우, 데이터의 무결성을 위해 중복된 항목은 자동으로 제거됩니다.
- **날짜 오류 처리**: 날짜 형식이 잘못된 데이터는 필터링 과정에서 자동으로 제외될 수 있으니 원본 데이터를 확인해 주세요.
- **보안**: 본 프로그램은 데이터를 외부 서버로 전송하지 않으며, 로컬 환경에서만 처리됩니다.
